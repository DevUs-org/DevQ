qrun \<filename\>: Runs the QASM file and submits its job to the hardware</br>

qps: Shows current jobs</br>

exit: Exits DevQ shell</br>

!!: Reruns previous command</br>

qtopology \<Qubit list\>?: Without arguments, dispplays each qubit connection. With qubit list argument, only displays the topology of the requested qubits.

qmem: Displays free and available qubits.

qmap \<Job id\>: Dispplays v2p map for the specified job id.

qerrors: Displays available Qubits and their error rates
